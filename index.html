Geoscience Python & ML Curriculum

Python programming, data science, and machine learning for geoscientists — Year 1 undergraduate through postgraduate specialization

Year 1

Foundations of Computational Geoscience

Introducing programming and data literacy to freshman geology and geophysics students with no prior coding experience

GEO 101
Introduction to Python for Geoscientists
6 weeks · 3 hrs/week Introductory
⌄
Prerequisites
None. Basic high school mathematics and curiosity are sufficient.

Learning outcomes

Write and run Python scripts using variables, loops, conditionals and functions
Read and write CSV and text files containing geological data
Use NumPy for basic numerical operations on geoscience datasets
Create simple line plots, scatter plots, and histograms of field data
Navigate the Jupyter Notebook environment effectively

Topics covered

  • Python syntax, data types, lists, dictionaries
  • Control flow and writing reusable functions
  • File I/O with CSV, TXT formats
  • NumPy arrays and basic statistics
  • Matplotlib: plotting geoscience measurements
  • Jupyter Notebook workflow
  • Worked examples: grain size analysis, temperature logs

Software & tools

Python 3.x Jupyter Notebook NumPy Matplotlib Pandas (intro) Anaconda Google Colab
GEO 102
Earth Data Literacy & Visualization
6 weeks · 3 hrs/week Introductory
⌄
Prerequisites
GEO 101 or basic familiarity with Python variables and loops.

Learning outcomes

Understand types of geoscience data: time series, spatial, borehole, image
Perform exploratory data analysis (EDA) on real geological datasets
Produce publication-quality geological plots using Matplotlib/Seaborn
Understand coordinate systems and basic map projections

Topics covered

  • Pandas DataFrames: loading and cleaning datasets
  • Descriptive statistics for geological data
  • Stratigraphic column plots, rose diagrams
  • Seaborn for statistical visualization
  • Introduction to geographic coordinate systems
  • Plotting on simple maps with GeoPandas

Software & tools

Pandas Seaborn Matplotlib GeoPandas (intro) Jupyter
Year 2

Applied Data Science for Geoscience

Building practical data science skills applied to geological, geophysical and environmental problems

GEO 201
Scientific Computing & Numerical Methods in Geoscience
8 weeks · 3 hrs/week Intermediate
⌄
Prerequisites
GEO 101, GEO 102. Concurrent enrollment in calculus or having completed high school calculus recommended.

Learning outcomes

Solve systems of equations arising in geophysical modelling
Apply numerical differentiation, integration and interpolation to field data
Implement finite difference methods for simple 1D heat/diffusion problems
Handle and interpolate irregularly spaced geoscience data

Topics covered

  • SciPy: integration, optimization, interpolation
  • Linear algebra with NumPy: inversion basics
  • Finite differences: heat flow and diffusion
  • Time series analysis: filtering, FFT, spectral methods
  • Gridding and contouring spatial data
  • Error analysis and uncertainty propagation

Software & tools

NumPySciPy MatplotlibJupyter SymPy

⌄
Prerequisites
GEO 102. Familiarity with map reading and coordinate systems helpful.

Learning outcomes

  • Vector data: shapefiles, GeoJSON with GeoPandas

  • Terrain analysis: slope, aspect, hillshade

GEO 203
Statistics & Geostatistics for Earth Scientists
8 weeks · 3 hrs/week Intermediate
⌄
Prerequisites
GEO 102, introductory statistics (or concurrent enrollment). GEO 201 recommended.

Learning outcomes

Apply classical statistical tests to geochemical and petrophysical data
Understand spatial autocorrelation and variogram analysis
Perform kriging interpolation of reservoir or geochemical data
Conduct Monte Carlo simulations for uncertainty estimation

Topics covered

  • Hypothesis testing, confidence intervals, ANOVA
  • Compositional data (Aitchison log-ratio methods)
  • Variogram modelling and spatial correlation
  • Ordinary and indicator kriging
  • Monte Carlo methods in resource estimation
  • Probability distributions relevant to geoscience
  • Regression and correlation analysis

Year 3

Machine Learning & Domain-Specific Applications

Applying ML methods to real geoscience workflows: lithology prediction, seismic interpretation, climate data, and subsurface modelling

GEO 301
Machine Learning for Geoscientists
10 weeks · 4 hrs/week Intermediate–Advanced
⌄
Prerequisites
GEO 201, GEO 203. Familiarity with linear algebra and probability. Python programming proficiency required.

Learning outcomes

Apply supervised ML models to lithology, facies, and mineral classification
Use unsupervised clustering to identify geological patterns in multivariate data
Evaluate model performance rigorously using cross-validation strategies
Interpret and communicate ML predictions with uncertainty quantification
Apply dimensionality reduction to well-log and geochemical datasets

Topics covered

  • ML workflow: data prep, feature engineering, validation
  • Supervised: Random Forest, SVM, Gradient Boosting
  • Unsupervised: K-Means, DBSCAN, hierarchical clustering
  • Dimensionality reduction: PCA, t-SNE, UMAP
  • Well-log facies classification (SEG ML contest)
  • Geochemical anomaly detection
  • Feature importance and SHAP values for interpretability
  • Dealing with class imbalance in geological data

Software & tools

Scikit-learnXGBoost LightGBMSHAP UMAP-learnImbalanced-learn WellyLasIO

Year 4 / Honours

Deep Learning & Advanced Geoscience Workflows

Neural networks, computer vision for geological imagery, subsurface modelling, and research-level software engineering

GEO 401
Deep Learning for Geoscience Applications
12 weeks · 5 hrs/week Advanced
⌄
Prerequisites
GEO 301 (ML fundamentals). Solid Python skills. Linear algebra and calculus. GPU access recommended (Google Colab Pro or local NVIDIA GPU).

Learning outcomes

Design, train and evaluate CNNs for geological image analysis
Apply transfer learning with pre-trained models to geoscience imagery
Implement recurrent networks and Transformers for sequence-type geoscience data
Use physics-informed neural networks for forward modelling
Apply generative models for synthetic data augmentation

Topics covered

  • Neural network fundamentals: backpropagation, optimizers
  • CNNs: architecture, convolutions, pooling
  • Transfer learning: ResNet, EfficientNet on thin sections
  • Object detection for geological features (YOLO, Faster RCNN)
  • LSTMs and Transformers for well-log prediction
  • Physics-Informed Neural Networks (PINNs) for PDEs
  • Autoencoders for seismic noise suppression
  • GANs for synthetic seismic/core image generation

Postgraduate

Specialised & Research-Level Training

Discipline-specific advanced courses for MSc and PhD students in petroleum geoscience, hydrogeology, geohazards, economic geology, and geodynamics

PGS 501
ML for Reservoir Characterization & Petrophysics
10 weeks · 5 hrs/week Postgraduate
⌄
Prerequisites
GEO 301 (ML fundamentals) or equivalent. Background in petrophysics or reservoir geology. Python proficiency.

Learning outcomes

Predict lithology, porosity, permeability and fluid saturation from well logs using ML
Integrate core data, well logs and seismic for ML-driven facies modelling
Apply deep learning to core image analysis for texture and fracture detection
Quantify prediction uncertainty for reservoir property estimation

Topics covered

  • Petrophysical log interpretation automation
  • Bayesian approaches to log-derived property prediction
  • CNN analysis of core CT and thin section images
  • Seismic inversion with neural networks
  • Data integration: well-to-seismic tie workflows
  • Stochastic simulation for reservoir modelling
  • Production data analytics and decline curve ML

PGS 506
Geodynamics & Tectonic Modelling with Python
10 weeks · 4 hrs/week Postgraduate
⌄
Prerequisites
GEO 201 (numerical methods), GEO 402. Solid mechanics, continuum mechanics, or geodynamics background. Advanced calculus.

Learning outcomes

Implement finite element and finite difference models for mantle convection and lithospheric flexure
Apply ML surrogates to accelerate computationally expensive geodynamic simulations
Infer tectonic parameters from geodetic, seismic and topographic datasets using Bayesian methods
Couple surface process models with tectonic uplift models

Topics covered

  • Finite element modelling with Underworld2 / FEniCS
  • Plate kinematic reconstructions with PyGPlates
  • Thermochronology data analysis and modelling (PECUBE)
  • Bayesian inference for geodynamic parameter estimation
  • Landscape evolution modelling with Landlab / Badlands
  • Emulation of expensive models with Gaussian Processes
